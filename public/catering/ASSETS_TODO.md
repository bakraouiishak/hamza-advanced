# Catering — Stock assets to download manually

The catering sub-site references these media paths. The site renders without them
(the `src` attributes simply 404), but for production you should drop these in.
Suggested free sources: **Pexels**, **Unsplash**, **Pixabay** (all royalty-free
for commercial use). Search keywords below are tuned to Saudi / Khaleeji cuisine
where possible.

## /hero/
- `catering-hero.mp4` — looping b-roll of plated dishes / chef hands.
  Pexels search: `catering buffet`, `chef plating`, `food preparation`. 10-20s,
  silent, 1080p.

## /services/
- `institutional.jpg` — workers / corporate cafeteria scene.
  Search: `corporate catering`, `cafeteria meal`.
- `buffet.jpg` — long buffet line with chafing dishes.
  Search: `event buffet table`, `wedding buffet`.
- `daily.jpg` — packaged lunch boxes / meal kits.
  Search: `packaged meal box`, `lunch box catering`.
- `menu.jpg` — chef writing / planning menu, ingredients board.
  Search: `chef menu planning`, `recipe notebook ingredients`.

## /menu/
- `kabsa.jpg` — Saudi kabsa with lamb. Search: `kabsa rice`, `saudi rice lamb`.
- `mandi.jpg` — mandi platter. Search: `mandi lamb`, `arabic rice dish`.
- `mezze.jpg` — Levantine mezze spread. Search: `mezze platter`, `hummus tabbouleh`.
- `grill.jpg` — open-flame mixed grill. Search: `mixed grill skewers`,
  `arabic grill platter`.
- `sweets.jpg` — kunafa / Arabic sweets. Search: `kunafa`, `arabic sweets platter`.
- `coffee.jpg` — Arabic coffee + dates. Search: `arabic coffee dates`,
  `gahwa dallah`.

## /chefs/
- `chef-1.jpg` — Arab male chef, white uniform, portrait. Search: `chef portrait`,
  `male chef uniform`. Choose images with clean dark backgrounds for the
  navy-themed cards.
- `chef-2.jpg` — second chef, similar framing.
- `chef-3.jpg` — pastry chef portrait, ideally with sweets in foreground.

Image dimensions: 1200×1500 (4:5) renders sharply on the card aspect ratios.
JPG quality 80 is enough; total weight target < 200 KB per image.
